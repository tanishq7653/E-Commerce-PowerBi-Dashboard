🚀 Project Overview

This project is an interactive Sales Analytics Dashboard built using Microsoft Power BI. The dashboard helps business owners analyze sales performance, profit trends, customer behavior, and product category performance through dynamic visualizations.

The project demonstrates the complete Business Intelligence workflow:

Data Import
Data Cleaning & Transformation (Power Query)
Data Modeling
DAX Calculations
Dashboard Design
Business Insights Generation

🛠️ Tech Stack
Power BI Desktop
Power Query
DAX (Data Analysis Expressions)
CSV Dataset
Data Modeling

📂 Dataset

The dataset contains Madhav Store sales transactions including:

Order ID
Customer Name
State
Category
Sub-Category
Payment Mode
Quantity
Sales Amount
Profit
Order Date





📈 Dashboard Features
Executive KPIs
Total Sales Amount
Total Profit
Total Quantity Sold
Average Order Value (AOV)
Interactive Visualizations
Sales by State
Profit by Month
Quantity by Category
Profit by Sub-Category
Sales by Customer
Payment Mode Distribution
Filters & Slicers
Quarter Filter
State Filter
Dynamic Dashboard Interaction
